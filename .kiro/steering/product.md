# Product Overview

React Design Playground is an interactive browser-based code editor for React components. It allows users to write JSX code in a live editor and see the rendered output in real-time.

Key features:
- Split-pane interface with code editor (left) and live preview canvas (right)
- Dynamic JSX compilation using Babel Standalone in the browser
- Error boundary system to catch and display compilation and runtime errors gracefully
- Execution history console for tracking code runs and debugging
- Default template featuring an architectural design configurator (Najdi and Modern themes)
- No build step required for user code - instant feedback loop

The playground is designed for rapid prototyping, learning React, and experimenting with component designs without needing a full development environment setup.
