# Product Overview

React Playground is an in-browser React component sandbox that compiles and renders React/JSX code in real-time without requiring a build step. It enables developers to quickly prototype and test React components with live preview.

## Core Features

- Real-time JSX/TypeScript compilation using Babel Standalone
- Iframe-based sandboxed component rendering
- Automatic library mocking (icons, Framer Motion, etc.)
- Template gallery with pre-built examples
- Execution history and error overlay
- Tailwind CSS support for styling
- Multi-line import parsing and handling

## Use Cases

- Rapid React component prototyping
- Testing AI-generated React components
- Learning and experimenting with React patterns
- Demonstrating React concepts without setup overhead
